What is Springboot?

Spring Boot is an extension of the Spring Framework that simplifies the process of building production-grade Spring-based applications with minimal configuration. It eliminates the need for complex XML configurations and boilerplate code by providing a set of pre-configured defaults and an embedded server (like Tomcat), allowing developers to run applications independently as standalone Java programs. With features like auto-configuration, starter dependencies, and production-ready tools, Spring Boot significantly speeds up development and deployment. In this project, Spring Boot was used to build the backend layer of the application, where RESTful APIs were developed to handle CRUD operations and communicate with the frontend. The use of annotations such as @SpringBootApplication, @RestController, @GetMapping, @PostMapping, and @Autowired made the code more readable and manageable. Spring Boot also integrates smoothly with Spring Data JPA, enabling seamless database interactions using repositories and simplifying ORM tasks with Hibernate. Exception handling, data validation, and logging were efficiently managed through built-in support. The application was further secured using Spring Security to manage user authentication and role-based authorization.
